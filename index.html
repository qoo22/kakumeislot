遊び方 / 仕様
  • 通常時 大当り確率: 1/99
  • 9・7・5・3・1 のいずれかが揃うと 確変突入
  • 確変中の大当り確率: 1/20 (30回転 / 約75%継続)
  • 確変中に当選すると Vマークが点灯し、3つ揃うと 超革命モード
  • 超革命: 1/2×3回転 + ラストチャンス1回。当選で必ず0回転に戻ってループ。1/2を4連続外すまで継続
  • 引き戻し: 確変終了後50回転以内に当選するとVマーク+1個追加で確変復活 (V2 → +1で超革命直行)
  • クレジット: 500スタート、BET 1〜20、当選で増加。端末ごとにlocalStorage保存。機械割120-130%調整
  • 連チャン1000枚突破ごとに「○○○○枚 OVER!!」カットイン
  • 超革命終了時は連チャン数+獲得枚数+称号のリザルト画面表示
  • 確変BGM: ラグタイム
  • 超革命BGM (4声=メロディ+ハーモニー+ベース+パッド構成、10連ごとに切替・70連でループ): 0-9連 第九1楽章 / 10-19連 カノン / 20-29連 アメージンググレース / 30-39連 運命1楽章 / 40-49連 歓喜の歌 / 50-59連 くるみ割り人形 / 60-69連 アイネクライネナハトムジーク。70連目で第九に戻りループ
  • 楽器音色: ストリングス/ブラス/合唱/オルガン/チェレスタ/ピチカートをADSRエンベロープと倍音合成で再現
  • 超革命の連チャン数は確変中の連チャン数を引き継いで継続
  • リーチ信頼度: シングル ★☆☆ / ダブル ★★☆ / トリプル ★★★ (約97%当選)
  • 「7」テンパイは激熱専用カットイン発生(信頼度ほぼ100%)。複数ラインリーチと複合で「超激熱！！」
  • 右上の「🧪 TEST 超革命」ボタンで超革命モードの演出をテスト試遊可能(クレジット・記録に影響なし)
  • 通常時の単発当選(偶数)が25%で 奇数昇格(勇者出現で図柄が斬られて奇数化→確変直行)
  • 確変・超革命中の当選は基本全て奇数図柄
  • 確変・超革命中はベット変更不可+自動回転 (確変/超革命終了で自動解除)
  • 「○○○枚 OVER!!」カットインは累計払出が1000/2000/5000/10000/50000/100000枚...と段階拡大刻みで発生 (確変→超革命を通して継続)
  • クレジット残高に応じてMAXベットが段階的に解放: 〜500/2000/10000/50000/200000/1000000で MAX 5/10/20/50/100/200/500
  • 全モードでバトル演出 → 勝利で当選確定
  • バトル中、稀に絶望からの覚醒/お助けキャラ参戦で逆転勝利
  • 超革命中、稀にハズレから復活当選
  • 左下のGOGO!ランプ: 当選確定時 35% でペカッ!と先告知点灯。確変中は青く点滅、超革命中は紫ネオンで高速点滅
  • 右上の RANKING ボタンで連チャン記録を表示 (スマホ対応)
  • キーボード: SPACE = SPIN, ENTER = AUTO切替
